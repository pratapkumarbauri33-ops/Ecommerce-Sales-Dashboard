## Ecommerce Sales Dashboard 📈🛒

This project focuses on analyzing ecommerce sales data using Excel dashboards and visualization techniques. The dashboard provides insights into sales performance, customer behavior, profit trends, and product category analysis.

## 🔍 Project Objectives

Analyze overall sales and profit trends
Identify top-performing products and categories
Track customer purchasing patterns
Compare regional sales performance
Visualize monthly and yearly sales growth
Create an interactive business dashboard

## 📁 Project Contents

- Raw Sales Dataset
- KPI Analysis
- Interactive Dashboard
- Category-wise Analysis
- Profit & Revenue Insights
- Regional Sales Analysis

## 📊 Tools Used

- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Data Cleaning
- Dashboard Design

## 📈 Key Insights

- Best-selling products and categories
- Sales and profit comparison
- Regional performance analysis
- Customer purchasing behavior
- Monthly sales trends

## 🎯 Outcome

This project helps businesses understand sales performance, customer trends, and profitability for better business decision making.
